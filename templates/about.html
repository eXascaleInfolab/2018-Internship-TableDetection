About.

This app implements 3 main steps:

  1. Crawling a domain and retrieving all PDF that can be found
  2. Analysing the PDF's found and detecting the tables present
  3. Extracting the data from the tables
How this tool works:
  1. Chose which domain to crawl. If you enter a specific url the crawler will start to crawl from there.
  2. After finishing fetching the pdf or being interrupted this tool will perform table detection on the pdf.
  3. Finally you will be able to see the collected data on the Stats page.
